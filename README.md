# Computer Concept CV Bot 🚀

A Python-based Telegram bot designed to provide a premium, service-oriented CV creation experience. Built with the `python-telegram-bot` framework and an SQLite database.

## 🌟 Features
- **Mandatory Payment Gateway:** Enforces a 20 Birr Telebirr payment via receipt upload before unlocking the bot.
- **Admin Approval Workflow:** Secure admin-side verification of payments.
- **8-Step Data Collection:** Seamlessly collects Basic Info, Education, Experience, Soft Skills, and optional Portraits/Reference PDFs.
- **Admin Console:** Global commands for `/stats` and `/broadcast`.
- **Direct Messaging:** Admin can reply directly to bot alerts to send finalized PDF documents securely back to individual clients.

## 🛠️ Built With
- Python 3.x
- python-telegram-bot (v20+)
- SQLite3
